Documentación del ejercicio Visual Odometry

Una vez finalizado el desarrollo del ejercicio de Visual Odometry, el siguiente paso fue preparar toda la documentación necesaria para que cualquier usuario pueda comprender su funcionamiento e implementar su propia solución. El objetivo era que el ejercicio no solo estuviera completamente desarrollado, sino que además contara con una guía clara que facilitase su utilización dentro de Robotics Academy Premium.

Documentación de usuario

Durante esta semana se elaboró la documentación oficial del ejercicio, que se encuentra disponible en la página de Robotics Academy. En ella se explica el objetivo del ejercicio, la interfaz disponible para el usuario y las funciones que proporciona la infraestructura para desarrollar el algoritmo de Visual Odometry.

La documentación puede consultarse en el siguiente enlace:

https://jderobot.github.io/RoboticsAcademy/exercises/ComputerVision/visual_odom

Contenido de la guía

La documentación comienza introduciendo el problema de la odometría visual y el objetivo del ejercicio, explicando cómo estimar el movimiento de una cámara a partir de una secuencia de imágenes. Posteriormente se describen las APIs disponibles para acceder a las imágenes, enviar la pose estimada y controlar el ritmo de ejecución del algoritmo.

También se dedica una parte importante a explicar el funcionamiento de los visores desarrollados durante estas semanas. Se muestra cómo visualizar la trayectoria estimada por el usuario, cómo interpretar la representación del Ground Truth cuando está disponible y cómo utilizar estas herramientas para depurar y evaluar el comportamiento del algoritmo.

Además, la guía describe los dos modos de funcionamiento del ejercicio:

  • Modo vídeo, pensado para trabajar únicamente con una secuencia de imágenes.
  • Modo rosbag, que permite reproducir tanto las imágenes como el Ground Truth sincronizado cuando este está disponible.

Finalmente, la documentación incluye una breve introducción teórica sobre los principales conceptos de Visual Odometry y proporciona diferentes recomendaciones para ayudar a los alumnos durante el desarrollo de su solución.

Vídeo demostrativo

Como complemento a la documentación escrita, también se preparó un vídeo demostrativo del ejercicio. Este vídeo muestra el funcionamiento general de la infraestructura, los distintos visores desarrollados y el comportamiento del ejercicio durante la ejecución de una solución de ejemplo.

El vídeo fue publicado en el canal de YouTube de Robotics Academy para acompañar a la documentación oficial y servir como una referencia visual para los futuros usuarios del ejercicio. Además, también se incorpora al blog para facilitar su consulta desde esta memoria de prácticas.

Conclusión

La preparación de esta documentación supone el último paso para completar el ejercicio de Visual Odometry. Gracias a ella, cualquier usuario dispone de una guía completa para comprender el funcionamiento del ejercicio, utilizar las herramientas proporcionadas por la infraestructura e implementar su propio algoritmo de odometría visual. Junto con el vídeo demostrativo, constituye un recurso fundamental para facilitar el aprendizaje y la incorporación del ejercicio a Robotics Academy Premium.
