Finalización del ejercicio Visual Odometry 3D

Tras el trabajo realizado durante las semanas anteriores, esta semana se ha completado el desarrollo del nuevo ejercicio de Visual Odometry 3D para Robotics Academy Premium. Durante el proceso se han implementado tanto la infraestructura del ejercicio como los distintos visores, el soporte para Ground Truth y la compatibilidad con rosbags, dando lugar a una versión completamente funcional.

Con ello concluye el desarrollo del ejercicio, quedando preparado para su incorporación al catálogo de prácticas de Robotics Academy Premium una vez finalice el proceso de revisión.

Funcionamiento del ejercicio

El objetivo del ejercicio será que el usuario implemente su propio algoritmo de Visual Odometry para estimar el movimiento de una cámara a partir de una secuencia de imágenes. Durante la ejecución, el algoritmo irá calculando la pose estimada del vehículo y enviándola a la infraestructura, que la representará en tiempo real mediante una trayectoria de color rojo en el visor 3D.

El ejercicio ofrece dos formas distintas de proporcionar los datos de entrada:

  • Modo vídeo: pensado para una primera toma de contacto con el algoritmo. En este modo únicamente se dispone de la secuencia de imágenes, por lo que no existe información de Ground Truth para comparar los resultados.
  • Modo rosbag: además de las imágenes, el rosbag puede incorporar la trayectoria real del vehículo (Ground Truth). Esto permite visualizar simultáneamente la trayectoria estimada por el usuario y la trayectoria real, facilitando la evaluación del algoritmo desarrollado.

Resultado final

El ejercicio desarrollado incorpora todas las herramientas necesarias para que los usuarios puedan implementar y depurar su solución de Visual Odometry de una forma visual e intuitiva. Los diferentes visores permiten observar tanto la imagen procesada como la evolución de la trayectoria estimada y, cuando está disponible, compararla con el Ground Truth del vehículo.

Todo el trabajo realizado durante las últimas semanas ha permitido construir un ejercicio mucho más completo que la versión inicial, incorporando soporte para visualización 3D, reproducción mediante rosbags y una infraestructura flexible capaz de trabajar con distintos conjuntos de datos.

Envío a Robotics Academy Premium

Una vez finalizado el desarrollo, se ha preparado un Pull Request del ejercicio para el repositorio de Robotics Academy Premium. En este momento queda pendiente del proceso de revisión por parte del equipo antes de ser incorporado oficialmente al repositorio principal.

Puede consultarse la rama Visual Odometry de RA Premium en el siguiente enlace:

https://github.com/JdeRobot/RoboticsAcademy-premium/tree/visual_odometry

Conclusión

Con la finalización de este ejercicio concluye uno de los principales trabajos desarrollados durante las prácticas. El nuevo ejercicio de Visual Odometry 3D proporciona una base completa para que futuros usuarios puedan implementar, probar y evaluar sus propios algoritmos de odometría visual utilizando tanto secuencias de vídeo como rosbags con información de Ground Truth, ofreciendo una experiencia mucho más cercana a la que encontrarían en aplicaciones reales basadas en ROS.
