  • API de control del robot y del entorno WebGUI.
  • Conversión de sistemas de coordenadas.
  • Procesamiento de datos del sensor láser.
  • Fundamentos teóricos del Dynamic Window Approach.
  • Implementación del algoritmo paso a paso.
  • Funciones de depuración y visualización.

Aspectos clave del algoritmo

Uno de los puntos más importantes de la documentación es la explicación del funcionamiento del algoritmo DWA, donde el robot evalúa múltiples trayectorias simuladas en el espacio de velocidades para seleccionar la mejor acción posible en cada iteración.

El proceso se basa en tres ideas principales:

  • Generación de velocidades candidatas dentro de una ventana dinámica.
  • Simulación de trayectorias futuras para cada par (v, w).
  • Evaluación mediante una función de coste ponderada.

Este enfoque permite que el robot tome decisiones locales en tiempo real, adaptándose al entorno sin necesidad de un plan global previo.
