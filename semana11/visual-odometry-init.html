Introducción a Visual Odometry

Durante esta semana se comienza a trabajar en el ejercicio de Visual Odometry (VO), una técnica fundamental en robótica que permite estimar el movimiento de un robot a partir de imágenes capturadas por una cámara.

La idea principal es reconstruir el movimiento del robot en un entorno 3D utilizando únicamente información visual. Esto se hace estimando cómo cambia la posición de la cámara entre fotogramas consecutivos.

¿Qué es Visual Odometry?

Visual Odometry es un algoritmo que estima la trayectoria de un sistema móvil (como un robot o vehículo) analizando la secuencia de imágenes capturadas por una cámara.

En nuestro caso, el sistema trabaja en un entorno 3D donde:

  • La cámara actúa como sensor principal
  • Se estiman posiciones relativas entre frames
  • Se reconstruye la trayectoria del robot en el espacio
El objetivo final es representar en 3D tanto la trayectoria estimada por el usuario como la trayectoria real (ground truth), permitiendo comparar ambos resultados visualmente.

Representación en 3D

El sistema visualiza el movimiento del robot en un entorno tridimensional donde se muestran:

  • Ground Truth (GT): posición real del robot
  • Estimación del usuario: trayectoria calculada por el algoritmo de VO

Esta comparación permite evaluar la precisión del algoritmo implementado.

Entrada de datos

Primero, el objetivo es que el ejercicio trabaje con videos como entrada principal, lo que permite probar algoritmos de estimación sin necesidad de sensores reales.

A largo plazo, el objetivo es incorporar datasets como KITTI mediante rosbags, lo que permitirá disponer no solo de imágenes, sino también de la posición real del vehículo.

Base teórica

Este trabajo se basa en estudios previos como el Trabajo de Fin de Máster de Pablo Asensio (2023), donde se exploran técnicas avanzadas de odometría visual dentro del entorno de Robotics Academy.

Conclusión

Visual Odometry permite estimar el movimiento de un robot únicamente a partir de imágenes, lo que lo convierte en una técnica clave en robótica autónoma. Este ejercicio servirá como base para comparar trayectorias estimadas frente a la posición real y evolucionará hacia el uso de datasets reales como KITTI mediante rosbags.
